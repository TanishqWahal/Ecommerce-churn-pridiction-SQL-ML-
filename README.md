Built an end-to-end customer churn prediction pipeline using SQL, Python, EDA and Machine Learning on real e-commerce transactional data.

## Business Problem
Customer retention is critical for e-commerce platforms.  
The objective was to identify customers likely to churn and generate actionable retention insights.

## Tech Stack
- MySQL (Data extraction & transformation)
- Python (Pandas)
- Matplotlib (Visualization)
- Scikit-learn (Logistic Regression)
- Machine Learning

## Project Workflow
1. Imported raw data into MySQL
2. Built customer analytics table using SQL joins
3. Performed revenue distribution analysis
4. Created customer segmentation (Low / Medium / High)
5. Engineered churn risk features
6. Trained Logistic Regression model
7. Evaluated using Confusion Matrix & Accuracy
